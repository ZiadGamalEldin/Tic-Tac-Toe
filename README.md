# 🎮 Tic Tac Toe — Advanced WinForms Edition

A modern desktop implementation of the classic Tic Tac Toe game built with C# and Windows Forms, featuring multiple game modes, scalable board sizes, and adjustable AI difficulty levels.

---

## ✨ Features

- 👥 Local Multiplayer — Play against another player on the same device
- 🤖 Player vs Computer — Challenge the built-in AI opponent
- 🧠 Multiple Difficulty Levels — Easy, Medium, Hard
- 🔲 Variable Board Sizes — 3×3 (Classic), 4×4, 5×5
- 🔄 Round System — Play multiple rounds with score tracking
- 🖥️ Clean Desktop Interface — Smooth offline experience

---

## 🎯 Gameplay

Players take turns placing their symbol on the grid.

The objective is to align the required number of symbols horizontally, vertically, or diagonally before the opponent does.

Larger boards and higher difficulties introduce deeper strategy and longer matches.
